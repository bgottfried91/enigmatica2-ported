# E2 Worldgen Datapack

This datapack is loaded globally by Open Loader from `config/openloader/data`.

Use it for the 1.20.1 replacement of the original E2 CoFH World ore generation:

- `data/e2_worldgen/worldgen/configured_feature/`
- `data/e2_worldgen/worldgen/placed_feature/`
- `data/forge/biome_modifier/`

Keep mod-specific ore generation configs disabled or aligned when this datapack
owns a material's generation.
